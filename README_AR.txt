لعبة Runner Bridge Game

طريقة البناء على Codemagic:
1. ارفع كل محتويات هذا المجلد على GitHub.
2. افتح Codemagic واختر المستودع.
3. مهم: اضغط Switch to YAML configuration.
4. اختر workflow باسم Android APK and AAB أو android-release.
5. اضغط Start build.
6. بعد النجاح ستجد APK و AAB في Artifacts.

اللعبة بدون إنترنت وبسيطة: لاعب يجري، وحش خلفه، ذهب، جسور، حبال، بحر، وعوائق.
